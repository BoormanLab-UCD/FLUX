Background

Evert-related fields (ERFs) are used to quantify the brain responses locked to a given stimulus. There is a large literature based EEG and MEG on respectively event-related potentials (ERPs) and event-related fields (ERFs) in relation various sensory and cognitive manipulations. In this section we will show how ERFs are estimated and plotted.

Preparing the data

Before initiating the ERF analysis read in the trials from the two FIFF files:
dataset{1,1}='training_raw-1.fif';
dataset{1,2}='training_raw-2.fif';
for i=1:2
cfg = [];
cfg.dataset = char(dataset{1,i});
cfg.trialfun = 'ft_trialfun_general';
cfg.trialdef.eventtype = 'STI101';
cfg.trialdef.eventvalue = [21 22];
cfg.trialdef.prestim = 2.5;
cfg.trialdef.poststim = 2;
cfg = ft_definetrial(cfg);
cfg.channel = {'MEGGRAD'};
cfg.demean = 'yes';
data(i,1) = ft_preprocessing(cfg);
end

Next 30 Hz lowpass filter will be applied. This is often done when calculating ERFs. The filtering will be done with padding ('zero-padding') the trial data segments to avoid having ringing artifact.
