Applying the MaxFilter after identifying bad sensors

MaxFilter is an Elekta/Neuromag software that was devloped to attenuate environmental and sensor noise in Elekta/Neuromag MEG systems. It is currently not possible to run Maxfilter via Fieldtrip or any other Matlab toolbox. It is however part of the MNE-Python package and to run Maxilter on the data with MNE-Python see the FLUX pipeline page: Artefact attenuation by MaxFilter - MNE-Python.
Alternatively, the stand-alone MaxFilter software is commonly accessible on sites with Elekta Neuromag MEG.
If MaxFilter is not available in either form, the subsequent Fieldtrip FLUX chapters can still be followed as the preprocessing and analysis will focus on MEG data recorded with gradiometer channels. However, we do not recommend the processing of magnetometers without MaxFiltering.
The tutorial dataset processed via MaxFilter is available at https://www.neuosc.com/flux/.
